# Medical-RAG-Agent-Side-Effect-Extraction-and-Summarization
An end-to-end Retrieval-Augmented Generation (RAG) system for extracting, validating, and synthesizing medication side-effects from real patient reviews using hybrid retrieval and agentic workflows.  Built using LangGraph, hybrid BM25 + dense retrieval, and instruction-tuned LLMs running fully locally in Google Colab.
